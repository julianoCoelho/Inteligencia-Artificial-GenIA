### 🚀 Aula 3: IA Avançada — RAG, Embeddings e Visão Computacional

- **Assuntos tratados:**
    
    - As limitações nativas das LLMs: Alucinação, _Knowledge Cutoff_ (data limite do conhecimento) e falta de acesso a dados privados.
        
    - **RAG (Retrieval-Augmented Generation):** Técnica que resolve esses problemas fornecendo uma base de dados externa para a IA consultar antes de responder.
        
    - **Embeddings:** Transformação de palavras/textos em vetores numéricos que representam seu significado semântico.
        
    - Bancos de Dados Vetoriais: Comparativo com o SQL tradicional e funcionamento da busca por similaridade.
        
    - Visão Computacional: Uma visão geral além do texto (detecção de objetos, segmentação, carros autônomos, diagnósticos médicos).
        
- **O que é importante:** Entender o ecossistema do RAG. É o padrão de mercado hoje para criar assistentes inteligentes corporativos que não inventam informações.
    
- **Onde ter atenção:** O conceito de **Embeddings** e matemática vetorial pode ser um pouco abstrato no início. Tenha muita atenção em como os textos são convertidos em números para que o banco vetorial encontre informações parecidas por "contexto" e não por palavra-chave exata.
